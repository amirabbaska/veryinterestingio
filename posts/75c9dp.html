    Q:

    How do blackouts work? Was the memory never recorded, never saved or deleted while sleeping?

    alnumero A:

    There are three stages to memory: encoding, storage, and retrieval. During a blackout from drinking, the prefrontal cortex is inhibited, which is responsible for a large portion of encoding events into short-term memory. Information must be encoded into short-term memory before it is stored into long-term memory. Therefore, because this information can’t be properly encoded and stored into short-term memory, the memory is unable to be processed into long-term memory via the hippocampus.

    Think of a livestream that’s not being saved.

    So basically the information is never recorded.

    GodBlessThisGhetto A:

    Interesting side note: there are two major sources of memory processes, the hippocampus and the amygdala. The hippocampus controls contextual memory processes while the amygdala controls cued memory processes.

    So in fear conditioning, you put a rat in a novel environment with a floor that you can electrify and you pair a tone with a mild electrical shock. This is very scary to the animal. The following day, you can put the animal back in the chamber and it will freeze, showing that it is afraid of the context in which the shock occurred. Then you can place the animal in the chamber after you have in some way modified the floor and walls. Now the animal will not freeze until you play the tone, so it's afraid of the cue that led to the shock.

    Now what's interesting is that alcohol effects the hippocampus and the amygdala differently. If you dose rats with alcohol, they will show issues with contextual memory, suggesting a hippocampus like deficit, but will show no loss of cued memory, indicating that the amygdala is not impaired.
