    ento5000 A:

    A straightforward explanation for colorization done manually (on a computer) -

    Black and white photos are considered grayscale, so all their pixels are shades of gray. Some programs detect how gray each pixel is and can apply a dynamic filter (in this example, let's say we wanted a red photo, so we apply a 100% red layer) based off of the gray underneath. Now the program calculates all 25% gray as 25% red, 70% gray to 70% red, and so on.

    You can make this filter multiple colors in different areas, so you eventually come out with a color photo.
