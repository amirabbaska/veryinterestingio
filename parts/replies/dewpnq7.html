    • r_matsuiya

      Is rTMS anything like electroshock therapy?

      • colonel_raleigh

        Basically it is electroshock, except that instead of applying electricity, the machine applies magnetic pulses, and then the brain responds by producing the electrical shock.

        The advantage is that rTMS only affects a very small area of the brain, whereas traditional electroshock affects the whole brain. So rTMS doesn't cause problems like memory loss. Both treatments can make you very tired, though.
