    • Midtek

      Start with x, a number normal (in base 10). Then replace all the 1's in the expansion with 2's to get the new number y. Then y is clearly not normal because its expansion contains no 1's.

      But y is also irrational. Why? A number is rational if and only if its expansion in all bases either terminates or repeats. If that were the case with y, then it would not be possible for x to have been normal. This is obvious if y has a terminating expansion. If y instead has a repeating expansion with period p (i.e., the length of the string that gets repeated is p), then strings of length p+1 cannot be uniformly distributed in x, contradicting normality.

      This solves the problem of /u/starbounder333's question of how a number could have a non-repeating expansion with no discernible pattern yet not be normal. All that is left is to prove the existence of some normal number x. (That's not easy if you want x to be normal in all bases, but if you just want x to be normal in base 10, then x = 123456789101112... (i.e., the concatenation of all natural numbers) will do the job.)

      Think of an infinite set of numbers. 1, 2, 3, 4..... Now remove the number 4. Anytime the number 4 shows up, skip it. This new set of numbers is also infinitely long. You will never count to the end of it. It is as long as the first set, but it does not contain a single 4

      Or think of the set of all even integers (whole numbers). It is infinite even though it only contains half of the possible integers.
