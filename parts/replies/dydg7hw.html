      You might enjoy reading about the Gingery Lathe, which is designed to be bootstrapped using only fairly basic hobbyist tools and materials.

      This link has some pictures, and there are a lot of videos of construction around on youtube.

      As you can probably imagine, we didn't suddenly construct a modern machine tool from nothing overnight. The first versions would have been crude and relatively imprecise, but each able to (with a lot of work, care and hand-fitting) produce a slightly improved version.

      Also, hand-work with simple tools like files, scrapers and laps can produce incredibly precise results if done properly, and with suitable measuring equipment. And an awful lot of that can be done with just a good surface plate and some gauge blocks.
