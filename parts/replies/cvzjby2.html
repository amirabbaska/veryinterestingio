    • MrJagaloon

      Text encoding works by assigning binary codes to unique characters. When software reads a stream of binary codes, each code is replaced by its unique character on the screen. If what you said were in place, it would change the basic principle of text encoding. Software would have to actually analyze the codes and implement logic to decode the binary codes. Also, this would increase the size of transmitted text and the time it takes to process the codes. It's not that what you suggested would be impossible, it's that it would defeat the purpose of encoding - to be fast and consistent.

      In my opinion, the reason text messages don't allow formatting is that it makes them look messy. Companies like Apple and Facebook put a lot of work into making their products clean and unified. They want every person to have the same experience on their products.
