    • TiZ_EX1

      I know that some of Linux or some Linux distros run on CLI, would a Linux user have to understand how to use command prompts in order to use Linux?

      Actually, all of Linux runs on CLI. However, there are graphical tools that interact with CLI tools, and vice versa. For example, there are graphical tools that automate and guide the use of the command line video tool, ffmpeg. Conversely, there are command line tools that can customize your graphical environment. Some distros vary in what graphical tools are available for certain CLI tools, but every distro has CLI underneath everything.

      Hope that helps.

    I use Linux (Peppermint) the most on my little old netbook, an Asus eeePC1000H. Does what I want, and has made a rather slow (1.6Ghz Atom from ~2010/11) machine usable for basic internet tasks even now, and even some light/retro gaming.

    This post has been what I've experienced. Don't be afraid to ask, don't be afraid to google, don't expect that things are going to Just Work. Go in with that mindset, and you shouldn't get too frustrated, and in time you'll learn what you need to in order to make it work.
