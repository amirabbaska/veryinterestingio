    Interestingly, Mac OS has a related reserved use for colon. In Classic Mac OS (anything pre-OSX) paths were comma colon separated and started with the drive name (For example, "MemoryBank:Users:foo:Desktop:bar.txt").

    OSX, as a Unix system, does allow colons in the filename, but to keep classic Mac OS users from getting confused the Finder will display the filename with a '/' in place of the ':'. So the filename "foo:bar" will be displayed as "foo/bar" in the Finder.
