  • pdpi

    However, the effect of this is negligible for Skyrim, because the program didn't really have a need for 64-bit arithmetic in the first place.

    In fact, naively replacing 32-bit arithmetic with 64-bit arithmetic all over the place would actually slow things down for no good reason.

    However, there is one specific place where recompiling a 32-bit program as 64-bit will net you a performance advantage: x86-64 adds some more general-purpose registers (basically, the absolute fastest type of storage in your computer, where the processor keeps the data it's working on for the fastest operation possible), which potentially means fewer memory accesses (likelier than not, just cache hits, but still more expensive than registers)
