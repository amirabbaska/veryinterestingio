    I think using network effects best explains the phenomenon here. With operating systems there are very strong networks effects: developers will only produce software for the most popular operating systems and hence we will naturally gravitate towards very few (or perhaps only one) operating system.

    With cars there are fewer networks effects, and so more competition is permitted. Hence we can have 5+ competitors. The only benefit now is from economies of scale, which are obviously limited at a certain value (I guess if there were no choice in types of cars then there would be more EoS possible and hence competition may be further reduced to a monopoly).
