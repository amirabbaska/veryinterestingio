    • mfb-

      OP said human habitation, that doesn't have to be outside.

      If you want to be outside, the first thing to worry about is the atmospheric composition. We can live in a large temperature range, but we need oxygen with at least ~10 kPa and at least ~15%, but not more than ~50 kPa. We need nitrogen to be below ~300 kPa and CO2 to be below something like 3 kPa. Various other gases shouldn't exceed various other limits either. It is unlikely that a random planet will satisfy all these conditions by chance, so we probably have to change the atmosphere a lot anyway - that includes the option to change its temperature.
