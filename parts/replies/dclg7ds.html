• Lavaah

    Locksmith here. To expand and correct on your information:

     

    Schlage has pin sizes ranging from 0 to 9 and most residential locks will all have 5 pins. The pin sizes for a Schlage lock are also standard in many other brands like original Baldwin hardware, Emtek, Ashley Norton, Rocky Mountain and more. There is also something called MACS (max adjacent cut speciation) that comes into play and it varies by key type. Schlage has a MACS of 5 so pins can be no more than a 5 size difference from each other. So you can not have a key like 91807

     

    The amount of possible keys changes by key type. Kwikset uses a different key but only has cuts 1-6 (technically they have a 7 depth but it is almost never used) with a MACS of 4.

     

    Also, when purchasing hardware from a department store depending on how they order it a set of locks will come in all on the same key. There is usually a keyed a like number on the box somewhere that would reference this. This is generally done in case someone is purchasing more than one for the same property.
