    • Beardhenge

      The problem is that the algorithm has to know when to stop, so it has to count how many digits it's already processed. The algorithm can forget the values of each subsequent digit, but it needs to know how many total digits to churn through before it burps out the nth output digit.

      If the total n number of digits is too high to store, the algorithm will fail. Since n can approach infinity, the algorithm is therefore impossible.
