  • dave_890

    When talking about a predicition, such as "percent growth", you use starting values:

    | (final - initial)| / initial

    |(4-3)|/3 = 1/3 = 33%

    Absolute value isn't necessarily required, as the sign of the result tells if you under- or overestimated.

    So, -33% = underestimated by 1/3; % error in the estimation was 25%.

  • Brunsy89

    This formula works in a scientific setting where the actual value (sometimes called the theoretical value) is the value that has been established by previous research. The experimental value (sometimes called the measured value) is the measured value obtained in an experiment.

    Since there is no previous research to establish the theoretical value (because the GDP was measured in the future and no previous research on the GDP of that year could have existed), there is no actual value in this particular scenario. The 4% is in fact the experimental value, because it was measured not already established.

    The percent error is completely relative to the wording of the question.
