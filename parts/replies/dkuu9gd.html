    An interesting side effect of dictionary compression is that if you compress a whole set of dictionaries for different languages i.e one zip file per language then you can use this to detect the language of an unknown phrase.

    If you add the mystery phrase to each of the zip files, the file that increase in size by the least amount is the correct language for the phrase. That's because this file will be the only one that already has the vocabulary of the phrase present so very little increase will take place.
