    • lorfeir

      What's happening in flash memory is basically static electricity, like when you rub a balloon against your hair.

      Flash memory uses a type of transistor called a "field effect transistor" or FET. FETs have three contacts: source, drain, and gate. The source and drain on the ones carrying the current to be controlled (when the power is on). Now, FETs come in different types, but for one particular type, there is a sort of barrier between the source and drain so no current can flow. The barrier can be removed if an electrostatic field is induced through it... and that is what the gate does. When a charge is applied to the gate, it creates an electrostatic field which moves the charges around the barrier, making it conductive... then current can flow between source and drain.

      For most uses of a FET (the majority of transistors in a computer are FETs now), the gate is directly connected to other devices, so the charge changes all the time, and so does the current between source and drain. In other words, the transistor turns on and off depending on the rest of the circuit. In flash memory, on the other hand, the gate isn't directly connected to anything. It's insulated from the rest of the circuit or is "floating" electrically. Any charge applied to the gate will remain for a long time, because there's really no way for the excess charge to flow away. So if a charge is somehow applied to the gate to turn the transistor on, the charge will still be there even after you have turned the device off. When you turn the device back on, the charge will still be there, and the FET will still conduct if the circuit sends current to it (that is, tries to read the state of the transistor).

      How does the charge get there in the first place? In a flash memory transistor, there is a fourth contact: a control gate. It's next to the regular gate but it's separated by a thin insulator. A strong electric field is applied to the control gate which then injects a charge across the insulator and into the floating gate. The process can be reversed to remove the charge so that either "1" or "0" can be written to the transistor.

      That's a rather long description, but believe it or not I'm skipping over a lot and generalizing a lot. A more detailed description can be found on Wikipedia, if you're so inclined.

      As an aside: if you've heard of EPROMs and EEPROMs, these work on the same general principle as Flash memory. They differ on several specific details, however, but concept of charge being stored on a gate is how they work at the basic level.
