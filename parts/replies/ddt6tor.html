      So modern computers are made with transistors. Basically, you can think of them as a valve in the middle of a pipe. When you input a 1 to the valve (tansistor) It opens and electricity flows past the valve. If you connect one end of the pipe to electricity, when you open the valve (transistor), electricity will flow out the other side. If you use this plumbing analogy, you can start to understand how you can make logical functions out switches.

      Now, you are thinking about moving wires, which is a very old way to do it. These days we have materials called semiconductors that open or close when you apply electricity to them. Essentially, it's an electronic switch with no moving parts.

      For instance if you put two valves on the same pipe one after the other, valve 1 AND valve 2 would have to be open for electricity to flow out the other end. This is a logical AND function that can be phrased as "If A=1 and B=1, then output 1, otherwise output 0". If you have two pipes each with one valve, and they connect with a Y into a single pipe, electricity flows out the end if either A OR B is open. This is another logical function that can be phrased as "if A=1 or B=1, then output 1, otherwise output 0". Now, you can make A and B statements, and get your answer of AND and OR with the result of the output.

      how do patterns of power/no power result in output

      So this is all very abstract, and you have to think hard about how a computer gets questions. So lets talk about how a computer could turn a C on a keyboard into binary, into a letter C on a screen.

      • You press the key on the keyboard
      • There is a series of switches that have the logical function of "If A is pressed, OR B is pressed, OR C ... OR Z is pressed, output 1"
      • That output is the on switch for the keyboard letter circuit
      • So since C was pressed, that is the 3rd letter, so the keyboard letter circuit gets 00000000000000000000000100 (3rd input is 1 for C)
      • The circuit then uses some fancy boolean logic with AND gates, OR gates, and inverters to turn that 26 letter string into 00000011, or binary for 3, which corresponds to C in our convention.
      • That circuit outputs a 1 to signal that it got a new input, and then outputs the code for the letter (binary 3).
      • That 1 signal is the on switch for the display circuit.
      • the display circuit then turns that binary 3 into the character C by turning on and off individual lights.

      So this was all how the idea of a computer works. It's just a lot of turning a 1 into a series of 1s and 0s that mean something unique.
