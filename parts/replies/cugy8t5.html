    Basically, when the machine is reset, the CPU begins executing code at a specific address, called a "reset vector" (https://en.wikipedia.org/wiki/Reset_vector). Every time you turn on the machine, it begins execution at the same address ... eventually loading a small program called a bootloader, which knows how to load the OS kernel.

    All of the microprocessor ICs in the computer (including the CPU) have a reset pin, which causes a hardware interrupt that sets the instruction pointer to the reset vector.
