    An example of a well defined use of "division by zero" is the dirac delta function. It is defined as the limit of e^-(t/a)^2 / (|a|* sqrt(pi)) as a goes to zero. We write it as δ(t). The useful property of this is that, while the "value" of the function at t=0 basically tends to +infinity from both sides, the integral (area under the curve) equals 1.

    It's used extensively in signal processing. If you want the area to change, you just multiply it by a value - so c*δ(t) has an area of c. You can also shift it (like any other function) so δ(t-t0) is centered at t0.

      • AbsentGlare

        You can think of the simpler version of the dirac function, actually it's not a function in the traditional sense of the word, it's more of a mathematical tool.

        The simpler version is a function that evaluates to zero everywhere, except on a triangle, centered at the origin, with base 2e and infinite height, where it is evaluated with the limit as e approaches zero. An integral of the dirac function becomes 1 if it contains the origin, zero otherwise. This is because the area under (1/2)(base)(height) = (1/2)(2e)(inf) = (limit)(1/2)(2)(0*inf) = 1.

        It allows us to do some interesting operations in linear systems.

        The discrete analog is the kronecker delta function, which has a value of 1 at the origin and evaluates to zero everywhere else. Very strange things happen when we sample something into the discrete domain.

  • Works_of_memercy

    The fundamental issue is that, once you divide by zero, you lose information.

    You mean, when you multiply by zero. You don't have to lose information when dividing by zero: the result is undefined in real arithmetic so you can define it however you want including without losing information, by saying that x / 0 = x * (1/0), thus generating an individual "infinite value" for each real number. So then (x / 0) * 0 = x roundtrips just fine.

    The problem is with multiplying by zero, because it's already defined to have a particular value, namely zero, and you can't redefine it to produce a bunch of differently flavored variations of zero that'd allow you to recover the original number.
