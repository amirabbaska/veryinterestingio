    Rotation isn't relative. Any point in a rotating body is accelerating towards it's centre. If you were in a closed room you wouldn't be able to tell if you were moving at constant velocity but you could tell if you were rotating. The actual forces felt would depend on how fast you were rotating of course, but you could detect it e.g. if you held up a pendulum it would only point directly down at the centre of the room. I've no idea how to this on a universal scale but the principle would be the same.
