      You're right to be confused! The definition of a factorial is "the product of all integers below and including the original number, until 1" e.g. 4! = 432*1. And it's this exact method we use to determine the number of arrangements of distinct items like cards. This method simply doesn't work for 0 items, nor negative, nor non-integer values, but the value of 0! is a common occurence in mathematics, and it'd be nice to have a value, so mathematicians have agreed to give it the value of 1 for two reasons.

      One reason is that it makes a lot of formulas just make sense. The mathematical exponential function e^x has the definition of e^x = 1 + x + x^2 /2! + x^3 /3! + ... You can also write this as the sum of all (x^n)/(n!) from n=0 to infinity. So the first term is x^0 /0! = 1/(0!), the second term is x^1 /1! = x, the third term is x^2 /(2!) = x^2 /2, etc. Looking back at the function, 1/(0!) should be equal to 1, so surely 0! = 1. It just makes sense, and a lot of formulas involving factorials just end up being very untidy if you keep making an exception for 0!, but in all of them, they'd work perfectly if 0!=1, so we just agree that 0! = 1 for tidiness' sake.

      The second reason is you can mathematically extend the factorial function. First of all, you can rewrite n! = n (n-1)! We know 1! = 1, so we can say 2! = 21! = 2. 3! = 3 2! = 6. 4! = 4 3! = 24 etc. This agrees with what we calculated with the other formula. We can use this fact to extend the function to 0, because it says that 1! = 1* 0!, so it makes sense that 0! = 1.

      More importantly however, because the factorial function is only really sensical for positive integers, mathematicians also found a new function called the gamma function that behaves identically for all positive integers, but also works for negative numbers and non-integers like 0.5. This means that the factorial function can really be thought of as a unique case of the more generalised gamma function, and the 'product of all integers ...' definition is just a neat little trick for when positive integer values are plugged into the gamma function. Sure enough, when we see what the value of the gamma function would be for 0 objects, we get 1 !

      TL;DR: A lot of maths points to 0! = 1. More convincingly, the factorial function is just a specific case of a more generalised Gamma function, which works for all real values, and it agrees that 0! = 1.
