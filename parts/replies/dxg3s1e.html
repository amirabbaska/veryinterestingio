      The beamsplitter does nothing but what it says: it gives each photon a 50/50 chance of passing through. What causes the phase shift is a difference in the path length that the photon goes down. In the experiments, the path length is modified to maximize the interference signal, in principle you could make it so the photons are perfectly in phase, in which case you see no interference pattern.

      The reason you see the opposite interference pattern at each detector is because the two entangled photons have opposite polarizations (call it 0° and 90°). At each detector we measure in a different basis (+45° or -45°) so we see an equal mixture of both polarizations of photons, and thus interference patterns. However, +45° and -45° are also orthogonal to each other, so they must sum to the same simple diffraction pattern that we see with no interference. Therefore the interference patterns measured in +45° and -45° must be perfectly out of phase. By cleverly picking which detectors measure in which polarization, we can work out the path information as is presented in those videos.
