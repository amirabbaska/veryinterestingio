      What the devkit for a console looks like really depends on what generation of console you're looking at. Today's consoles are basically the same as PCs so a dev system is just a high-end PC with a bit of custom hardware/software. For older systems, it would be some sort of expensive-as-fuck pre-production version of the hardware with some easy way of loading code onto it (rather than having to burn a ROM every time).
