    The classic game consists of a 4x4 grid.

    In a perfect game, you would lose the game by having these blocks in order:

    2, 4, 8, 16, ..., 2^(16)

    That is the case because you have no moves left, and no duplicate blocks during the loss.

    max_score = s(2) + s(4) + ... s(2^(16))

    The following recursion is the score that you get by assembling a 2^(x) block:

    s(2) = 0

    s(2^(x)) = 2 * s(2^(x-1)) + 2^x

    That is because you need 2 smaller blocks to get the next block.

    By writing a quick python script we get the max score of 1835012 3932100.
