      • -tp-

        No. A proof by contradiction will assume that there are only a finite number of primes and reach a contradiction. It is possible to prove it this way, I agree. But this is not how Euclid did it. He proved that given any finite set of primes, there is a prime outside this set. There is a prime outside every finite set of primes, and thus the set of primes is infinite. These finite sets are not assumed to contain all primes.
