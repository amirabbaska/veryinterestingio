    • woodbetween

      Basis vectors are different states. A particle's state "rotates" in the vector space as it evolves in time. Like a 3D vector, the particle state can be projected onto different axes. In QM, the projection can be complex. Any physical measurement will involve the projection times its complex conjugate, so the complex character never manifests physically. The particle state is a single vector in this space.

      Importantly, QM is a Hilbert space over C and not R, so a vector in this Hilbert space takes on complex values.

      If so, how do complex numbers come into it?

      In a euclidean vector space isn’t “enough space” for encoding dynamics. The square of the wave function is interpreted as a probability density of the particles position. Now the dynamics must preserve the norm of a state |U(t)ψ| = |ψ| = 1. Thus the time evolution operator U(t) must be unitary/orthogonal.

      To get a differential equation for the time evolution you need Stone’s theorem which requires a complex Hilbert space.

    • woodbetween

      In Dirac notation, a vector in Hilbert space is usually represented by by the notation |f>, which would be called a "ket". The inner product (or "bracket") is usually represented as <g|f>, which suggests the inner product is the ket |f> with the object <g|, which Dirac called a "bra". Bra+ket=bracket.

      The ket |f> corresponds to a generic vector in the Hilbert space. Physically, it represents some general particle state before measurement.

      Suppose, for simplicity, that there are four possible outcomes of a physical measurement. Suppose we're measuring the observable quantity X, and X can take the values a,b,c,d. Then there exist kets |a>, |b>, |c>, |d> which are vectors in Hilbert space that correspond to particles that give measurements X=a, etc. These form a basis. You can't actually measure X=a AND X=b at the same time, so the bases have to be orthgonal.

      For a general ket |f>, we can project it onto this basis with the dot product to get its components <a|f>, <b|f>, <c|f>, <d|f>, which for simplicity I'll call fa, fb, fc, fd. Then we can also write the general state |f>= fa|a> + fb|b> + fc|c> + fd|d>.

      The components fa, fb, fc, fd are in general complex, and they might be time-dependent, which means the vector might "rotate" in the Hilbert space over time.

      To find the probability that the general state |f> will give the measurement X=a, it is

      Pr(X=a) = <f|a><a|f> = (fa*) fa

      where fa* is the complex conjugate of fa, which ensure rhe probability will be positive and real.
