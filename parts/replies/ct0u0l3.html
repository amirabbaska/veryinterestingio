    SMTP was the problematic in the distance past because both sysadmins and software didn't check MX records correctly. When you send mail to x@x.com modern servers do a MX lookup of x.com which should return the address of a SMTP server. Even in the mid 90's this didn't always happen correctly and an A record lookup for x.com commonly occured. If your HTTP server and your SMTP shared the same address that was fine, if they didn't, there was problems.
