  • Procrasturbatization

    The best explanation IMO. In memory at least, it's nothing to do with computers "working in binary", and everything to do with the implications on the physical layout of memory using binary addresses. To use 1000 would leave 24 potential "boxes" that could be addressed, but are instead absent. In HDDs, manufacturers get away with using 1000 because their platters don't have the same constraint, so they just fit as much as they can and label it so it appears larger.
