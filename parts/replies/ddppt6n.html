    For the uninitiated, equation (5) expresses the proper time s (the time elapsed according to the accelerated observer) in terms of the coordinate time t as measured by the inertial observer. The equation is

    t = (c/a) sinh(as/c)

    where a = constant proper acceleration of the accelerated observer. (There is actually an error in the equation.) Since sinh(z) is a strictly increasing function of z with sinh(z) ---> infinity as z ---> infinity, we see that there is no such "asymptotic coordinate time" that the OP suggests. In other words, for each proper time elapsed there is a corresponding (finite) coordinate time. It's not like an object falling into a black hole or something where there is not a 1-1 correspondence between proper and coordinate time.

    However... you are going to have to wait a long time even for a rather small proper time. Note that sinh(z) ~ e^(z)/2 as z --> infinity. (Indeed, the relative difference between sinh(z) and e^(z)/2 is less than 1% for z > 2.5.) Conversely, note that s = (c/a) sinh^(-1)(at/c) ~ (c/a)ln(2at/c). For even a moderately large value of s, you need a very large value of t since the logarithm grows so slowly.
