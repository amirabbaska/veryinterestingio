    There are possible ways other than "it's alive" to tell if a lifeform is extraterrestrial. The ATCG code. If some other chemical was found to be used to code genetics. The chirality (handedness) of other life chemicals.

    It turns out all life on earth has the same forms of these factors. So if we found radically different forms, we can be reasonably sure they are in fact, extraterrestrial. If they were the same the results will be inconclusive and some other (more difficult) ways of determining non-earth forms would be needed.
