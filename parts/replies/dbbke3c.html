    • dale_glass

      I wouldn't say so. Many past frustrations are gone.

      • Spending hours freeing up that last KB of conventional memory is long past
      • So is worrying about having enough RAM at all. At 16GB I've yet to find something I need to do that would need more than that. If that ever happened, I can upgrade to 32GB.
      • So is waiting for things to download, in general. 56K was waiting for everything. 100Mbps is waiting for very select things. If I download a movie I can download it faster than I can watch it.
      • Downloads in general are quick and uneventful rather than stressful events where I might end up wasting 3 hours if it fails
      • The disk isn't slow anymore with SSD
      • Backups are far easier because I can use the network rather than tons of floppies/CDs/etc.

      So I would say the point is that things indeed get better.
