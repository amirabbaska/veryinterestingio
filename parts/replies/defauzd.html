    • Ping_and_Beers

      It is possible, but very difficult to accomplish depending on how isolated the VMs are. If the VMS are networked, then a virus can propagate from host to host as normal. If not, then the malware needs to be intentionally programmed to be virtualization "aware" and exploit the underlying hypervisor, and the ability to do that will also be dependent on whether you're using a type 1 or 2 hypervisor.
