    The speed also decreases when there's other devices on the same frequency, not necessarily connected to the router/tower and even if not following the same protocol.

    Each device listens to the frequency before trying to send something. If the frequency is busy, it waits a random amount of time to try again.

    When a given device is waiting for data, it actually tries to receive every transmission using the same frequency, coming from any device in range, but it can only decode the data meant for it.
