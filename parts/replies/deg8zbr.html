    • kaliena

      A behaviorlist would be best to consult. If this were me, on an isolated island etc -- I'd put this 'on queue' with positive reinforcement. Clicker and treats. Get it really enforced. Get it on stimuli control to where dog does it only when asked. Then never give command.

      In the middle I would teach a super strong default behavior that can not be executed while doing the undesired behavior. Maybe a 'go to bed' combined with 'head on ground'. This I would start super rewarding. Make it rain compared to the undesired behavior. You want both to be locked in and only offered on request -- but default should be the new OCD.
