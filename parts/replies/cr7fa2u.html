    If you are using Iphone, I'm afraid I can't help you. I am not familiar with blocking ads in iOS. Someone more qualified would be able to offer assistance.

    For Android

    If you are using an Android phone and rooted, then Adaway. Just Install it and do the initial update it'll run smooth. Don't ever worry about it again.


    Now if you are using an Android phone and it is not rooted, then Adblock Plus. This needs some configurations in the WiFi settings.

    Both these apps are not available through the Google play store. So you will have to download the apk packages and install them manually. This involves going to Security settings and allowing the 'Installation of apps from Unknown Sources'.

    Or you can install an alternate app store line F-droid, and then installing the apps through the store.

    Last Note: If you are using a phone like One plus One with Cyanogen mod on it, then it's already rooted (So just go ahead and install Adaway)

    Hope this helps.
