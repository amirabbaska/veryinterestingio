    • noncongruent

      Wire heating is the main issue in home wiring, it is what dictates minimum size wiring for given circuit ampacities. The only negatives with using larger wire sizes are the additional costs of the larger wire and the increased difficulty of working with thicker wire. There are also maximum sizes for the various terminals on things like switches, outlets, and breakers, so in your example of 2 gauge wire it would not be possible because it would not fit into the various terminals of fixtures typically used on 15 and 20 amp circuits.

      It is, however, considered "above code" to use one size larger wire for 15A outlet circuits. Code requires 14ga, so 12ga is considered better.

      All wires have a built in resistance. It is usually so low it isn't worried about. Resistance=(resistivity of material * length of cable)/cross section area. The thicker the cable the less resistance (more power if you have a set voltage less power if you have a set current)
