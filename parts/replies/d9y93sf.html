    To go down the rabbit hole a bit more, there is a universal turing machine, U. That is a "computer" which will simulate any other computer if given the right input. Since U is, itself a turing machine, there is an input for which U will simulate U.

    On the other hand, it should be emphasized (as you briefly mentioned) that no physical computer is actually a Turing machine, but, since they have finite memory, they are equivalent to finite automata. So there is no such thing as a physical "turing-complete computer". It is clear, that there is no universal finite automata.

    Thus, the answer to the OP's question is Yes for Turing machines (infinite memory), but No for physical computers.

    It would require an unfanthomably powerful computer which might not be possible to build in our universe, but as far as I know, we are not aware of any theory that precludes this.

    One of the interesting subtleties of this idea is that beings existing in a simulation wouldn't have a reference as to the "real" time they're living in. So, for instance, if a single second in the simulation took a million years to render in the "real" world, the beings inside the simulation would still experience it as a second.

    If the simulation can run forever, the computation power required to run the simulation reduces inversely.

  • jminuse

    According to our current understanding, physical reality could in principle be simulated on a computer.

    This is not true. Our best current models of reality (quantum mechanics and general relativity) are known to be incomplete. So, currently, we can't simulate any part of reality without fudging it. And there is no proof that the necessary complete model even exists, or if it does, that it is computable.
