    • gothcopter

      Absolutely it can be done. If you run two additional laps, then you only need run them at 4x the speed of the first lap to double your average. For example, let's say the laps are 100m and you ran the first one in 4 seconds. (My you're fast!) 100m/4s = 25m/s

      Now you run two additional laps at 4x that speed, which would be 100m/s. So each lap will take you 1 second to run. So the total time you spent running was 6 seconds. 300m/6s = 50m/s

      So you've doubled your average speed.

      You can work this out yourself with the v=d/t formula and some basic algebra.
