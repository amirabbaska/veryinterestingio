      A common misconception is that gifs can only contain 256 colours. This is almost true: a gif can only contain 256 colours per frame. Each individual frame, however, can have its own 8-bit (256 colour) colour palette. So true colour gifs are in fact possible by abusing this, and having a gif simply be a collection of frames to be overlaid, each with 255 unique colours. (One is lost to the required transparency.) Example.

      The result is in almost all cases however an ungodly large file, so you really should just use png for this.
