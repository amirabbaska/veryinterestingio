    Yup. It works in other bases.

    If the base is B, the difference between the number and it's "reverse" is always divisible by (B-1).

    You can always "decompose" or break apart a number minus its reverse into a sum of numbers minus their reverses where every single one is an integer multiple of B^n - 1 for some n.

    Then we simply note this:

    (B^n - 1) % (B-1) = 0

    why?

    (B^n - 1) = (B - 1)(B^(n-1) + B^(n-2) ... B^2 + B + 1)

    For example...

    2030 - 0302 = 2030 - 302
    = (2000 - 2) + (30 - 300)
    = 2(B^3 - 1) + 3(-B)(B^1 - 1)
