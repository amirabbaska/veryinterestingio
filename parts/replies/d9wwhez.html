  • Maximillord

    Apart from that, there are client side/server side codes which run in background during uploading process. You can compress the data chunk in client side before sending and reduce packet size to an extent. Or upload data chunks via different servers which gives you a much wider bandwidth. Then on the server side they get uncompressed and merged as a single file. These methods are generally used for big files because it requires lots of coding and complex server networks. Developers don't feel the need to do it for images.
