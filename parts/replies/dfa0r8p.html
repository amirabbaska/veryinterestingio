    This is true in isolation, but only explains things running slightly slower.

    In general, when you see a noticable slowdown it is bc you have apps at odds with each other.

    For example, two Antivirus products on the same system.

    AV1 sees AV2 is doing something. AV1: "I must check out this activity!"

    AV2: "there is new activity I must check!" (Which is AV1 checking AV2)

    Feedback loop started

    Add in other apps, and you now have these apps doing the electronic version of sniffing all the other dogs in a circle
