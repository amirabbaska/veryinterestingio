    the pupils serve the same function as the aperture on a camera, and there's the same tradeoff: if they're smaller, the image can be much sharper, but it will be dimmer and so you need a lot of light to justify a small pupil; if they're larger, the image will be blurrier but you can see even if light is very low.

    but the pupils are definitely a distant second when it comes to the light-regime transition mechanisms.
