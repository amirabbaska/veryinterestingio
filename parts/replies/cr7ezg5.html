  • wigchert

    Yep, this is exactly how most of those tools work. To add to that, what I haven't seen anyone mention yet is that Adblock blocks per CSS class and frame dimensions too. For example if a frame is 300px wide and 250px high, it will get blocked as well as elements with classes like '300x250-frame'. What these anti-Adblock tools do is add baiting classes to the site, if any of them aren't loaded, it means they are being blocked.
