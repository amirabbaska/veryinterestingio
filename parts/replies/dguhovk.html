      Given: f(x,y) := x/y

      • lim(x -> 0+, y -> 0+, f) = 1
      • lim(x -> 0+, y -> 0-, f) = -1
      • lim(x -> 0-, y -> 0+, f) = -1
      • lim(x -> 0-, y -> 0-, f) = 1
      • x := 0, lim(y -> 0, f) = 0

      Graph of the equation

      In English:

      As x approaches zero from the right and y approaches zero from the top, f(x,y) approaches one. As x approaches zero from the left and y approaches zero from the top, f(x,y) approaches negative one. Then I list the inverses of those limits. Lastly, if x is defined to be zero, the limit of f(x,y) as y approaches zero is zero.

      Zero divided by zero is undefined because there is not one distinct solution for the operation.
