    • IM_OK_AMA

      Two reasons, first consoles have CPUs also and those need to be emulated, which isn't a task that works well for the way GPUs think.

      Second because GPUs are set up kinda like mini-computers. They also have high speed cache and low speed RAM, so the same problem would apply even if you could convince your GPU to do the work.

    • iBoMbY

      The basic problem is knowledge: The stuff could probably run better, if you'd know all the details on both sides, but in most cases the people writing the emulators know only a fraction of what's possible from both sides, because they have to reverse engineer a lot of it, because it isn't documented, and even if it was, it would take a lot of time and effort.
