    • SexyMonad

      It isn't really teaching it math so much as faking it using logical operations. Transistors are combined to form logic gates. These are simple circuits that take usually one or two binary input wires and have an output wire.

      For example, the XOR gate (exclusive or) causes the output wire to be 1 whenever either input wire is 1. But if both inputs are 0 or both are 1, the output is zero. There are other kinds of gates that correspond to other logical operations.

      Now think of how binary math works for one bit. 0+0=0, 0+1=1, 1+0=1, and 1+1=0 (with a carry of 1). XOR is equivalent to single-bit addition!

      Now if you put 8 next to each other, you have 8-bit addition (almost... you need to add some more to handle the carry from right-to-left).

      Other operations vary in complexity and size. Some work with integers and others with floating point numbers.

    • SirButcher

      Electricity only has two stages: it is either ON (there is a voltage difference between two points) or it is OFF (no difference / no connection). So it would be pretty hard to encode other numbers: electricity can't be partly on or partly off.

      Not ELI5: In reality this is a little more complicated - as every passing electron has a magnetic field which can induce electricity in the surrounding wires they don't actually use ON - OFF but using HIGH and LOW values with a big PROHIBITED area between these two - so even if interference happen (which happen) the circuit will know if the signal is legit or not. This called digital signal, most often using 5V. And they don't use the OFF as a real off (breaking the circuit) because it is very hard to detect. Low voltage is easier to create and very easy to actually detect it. It could be possible to create other stages as well (other accepted voltage levels) but it would just complicate everything without any gains.

      Going even lower level: a circuit is actually made from transistors: which is basically an electrically operated switch. They use two different type of silicone sandwiched together like this: N - P - N. The P in the middle is blocking electricity. Each transistor has three legs: one input, one output and one which controls if the gate (the middle part of the sandwich) is opened or not. If the P part gets enough electricity it allows electrons to flow from the input to the output - the circuit is closed, the signal can travel. If the P part doesn't get enough, the gate remains closed, the signal can't move on. Imagine it as a light switch - you are the control. If you turn the switch on, the lamp get the signal. If not, then it remains dark. The transistors are the same, just MUCH-MUCH-MUCH smaller. And they, as I said, can be controlled by electricity - with the mechanical switch it would be impossible to do this fast or this small (and producing such a mechanical transistor would be much harder).
