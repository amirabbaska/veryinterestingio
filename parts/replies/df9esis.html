    • Why_is_that

      Yes. I think this "paradox" occurs due to how we imagine the reference frames. The photon travels a straight-line path but it also is traveling speed C (so this frame is a bit weird to think about). The lensing from a massive object will bend the photon's path and this is what we observer from the massive object and most frames you would consider outside the photon. This path is curved and the frame we are observing from is not at speed C, so it's a frame a little bit more relatable to. The photons that the massive object absorbs, are the actual straight-line paths an outside observer would see.
