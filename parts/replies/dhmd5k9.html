  • BlazeOrangeDeer

    QFT fits within the QM framework just fine. There is a wavefunction, it just has amplitudes for field configurations rather than particle positions. Entanglement has the same definition and the same properties, and you can ask exactly the same questions about when the state supposedly collapses.

    The short answer is, it appears to be collapsed whenever someone measures it, but it does so without regard to the time of measurement. This is because there isn't even a global clock time in QFT (since it is based on special relativity), or you could say there are many equally valid global clocks that each disagree about whether events even happen at the same time. In other words, in some cases there isn't even an answer to the question of which measurement was done first, so the results cannot depend on it.

    TL;DR entanglement doesn't care about time, it looks collapsed when you measure and that's all you can say
