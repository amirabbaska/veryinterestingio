• Mathematics How do we calculate numbers to an irrational exponent like x^pi?

    NEW
    Q:

    I always understood that a^(m/n) = a times a m number of times and then take the n^th root

    but for irrational exponents, we just have an infinite non-repeating decimal that doesn't fall into that form.
    I was thinking maybe for something like pi, calculators just round off somewhere and make the number something like 314159/100000 but I wasn't sure if it would overload the calculator first or something.

    either-way A:

    If you’re asking how exponentiation is defined on irrational numbers, it’s just what you might expect. For any irrational number, there’s only one value that makes the function continuous, or smooth, and that’s what it’s defined to be. To calculate it, what you said works. However, there isn’t even a very clean algorithm to compute nth roots. Since these don’t tend to be rational either, they must be approximated as well.

    crownmint A:

    From a pure maths point of view we might the number 2^pi as the limit of a sequence of approximations. For example we could take the sequence 2^3, 2^3.1, 2^3.14, ... Where each number in the sequence is a rational power of 2. I don't know how a calculator solves these problems (it's almost definitely more efficient than doing this) but that is one way of defining what the number is.

    Midtek A:

    We simply declare the function x^(y) to be continuous in each variable separately. So if you want to compute x^(e) for some fixed x, then find your favorite sequence rn of rational numbers that converges to e. The sequence x^(rn) is then perfectly well-defined, and we simply declare its limit to be x^(e).

    This is all possible by the least upper bound property of real numbers.

    ramennoodle A:

    On computers, it is done like this:

    x^pi = exp(ln(x^pi )) = exp(pi ln(x))
