    drafterman A:

    The most common percentage error formula is this:

    % Error = | actual value - predicted value | / actual value x 100

    In this case:

    % Error = | 0.04 - 0.03 | / 0.04 x 100
= | 0.01 | / 0.04 x 100
= 0.25 x 100
= 25%

    Pretty sure it's 25%

    Formula for percent error is

    |(actual value - experimental estimated value)| / actual value

    In this case the economist's estimate would serve as the experimental value, so

    |(4 - 3)| / 4 = .25

    KurtWagnerX A:

    Both of the following statement are mathematically accurate:

    Actual growth was 33% more than predicted.

    Predicted growth was 25% less than the actual growth.

    Here's another way of looking at it with just hard numbers:

    4 is about 33% more than 3.

    3 is 25% less than 4.

    BUT as others are pointing out, conventionally error is described as a percentage of actual results. The different was one pp, the actual was 4, so we call that an error of 25% even though it's also accurate to say growth was 33% more than predicted.

    BetaFisher A:

    Actuary here, common question.

    When talking about percentage change, you've got to provide context, and should avoid introducing a percentage that they're off.

    For example, what if real growth was 0.01%?, is the economist off by [(0.01% - 3.00%)] / (0.01%) = 29900%. That's not useful information.

    It's best to say what you said: 3% was predicted, actual was 4%. You will only ever introduce confusion by stating, "so they were off by 33%."
