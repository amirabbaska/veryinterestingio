There are plenty of operators that are associative, but not commutative. And plenty that are the other way around. Just to give one example of each:

Function composition: we define the composition of two functions from R->R, f * g as f(g(x)). Checking associativity:

(f g) h = f(g(x)) h = f(g(h(x)) = f g(h(x)) = f (g h)

However, this operator is not commutative. Let f(x) = x + 1 and g(x) = x^(2), then f g = x^2 + 1, but g f = (x + 1)^(2).

The other way around, consider the operator that takes 2 real numbers and returns the average. So a (+) b = (a + b) / 2. This operator is clearly commutative since addition is. But it is not associative. Take a = 0, b = 4, c = 2:

(a (+) b) (+) c = 2 (+) 2 = 2

a (+) (b (+) c) = 0 (+) 3 = 1.5

There are plenty of other examples that you can find for either direction, so in general it is definitely not the case that associativity and commutativity are interchangeable.

Another example would be matrix multiplication. [A] times [B] doesn't necessarily equal [B] times [A].

Believe it or not but you'll learn in any introductory abstract algebra course that the Associative property is more common and general than the Commutative property.

A good example of non-associative (but extremely useful) operation that you've likely seen before is the cross product x on vectors in 3-dimensional space. You usually see this first as a way to take two vectors u and v, and form a new vector (u x v) which is orthogonal to the original two vectors.

This operation is anti-commutative, i.e., it commutes up to a minus sign, since (u x v) = - (v x u).

It's very non-associative though. Instead of associativity, it satisfies the "Jacobi Identity", which can be stated in the following form. Given three vectors a, b, and c, it's always true that:

a x (b x c) = (a x b) x c + b x (a x c)

This identity is important in defining an algebraic object called Lie algebras. Beyond being fascinating objects from a mathematical standpoint, Lie algebras also have important applications to particle physics.

If we work with vectors with complex coordinates (i.e., we don't mind giving -1 a square root), the vectors in 3-dimensional space with the cross product form the Lie algebra sl_2(C), which is (arguably!) the most important Lie algebra of all!
