This isn't possible with passive optics. You can see this by appealing to entropy. Unpolarized light is an example of a mixed state, while polarized light, whether linear or circular, is a pure state. The entropy of a pure state is always lower than the entropy for a mixed state.

So to go from unpolarized light (a mixed state) to polarized light (a pure state) at the same intensity, you would need to decrease entropy.

That's a good question! I don't know myself whether a practical setup exists, but at least in principle, it should be possible.

One theoretical way to do it is to use a polarizing beam splitter which is (more or less) lossless, like the Nicol prism, and then use the Faraday effect to rotate the polarization of the other before finally combining them again. But this seems rather impractical.
