Scientomancer A:

So let's look at an equation like 3 • 5 = 15. Easy. Now watch how we can take that equation and turn it into 3 = 15 ÷ 5. Also easy.

Now let's put zero in the mix. 3 • 0 = 0. Okay. So then it would follow that 3 = 0 ÷ 0. And if we change 3 to 4, we would get 4 • 0 = 0 and 4 = 0 ÷ 0.

Division by 0 is undefined because it isn't equal to any one value. It could be equivalent to anything we like using the above reasoning. This is a problem because the arithmetic operations are actually functions like you learn about in algebra, and functions are only allowed to have one single output for any given input.

Birdy1072 A:

You have to remember that zero isn't just a number, it represents "something". Or in this case, zero represents nothing and you can't divide by nothing. If you tried to divide 1 by 0, that's impossible because you can't divide by something that isn't there. Same goes for when you try to divide 0 by 0.

Another example. Let's say you have four apples. You have to define how you're going to divide up those apples. You can divide by two, so you'll have two groups, but you can't have no groups at all -- you'll always have at least 1 group with 4 apples in it.

Now, you can divide 0 by 1 because you can divide nothing up however many times you want but you'll still have nothing.
