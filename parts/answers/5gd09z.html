I am assuming you are referring to math problems which have not been solved, but may be solved in the future like the and Riemann Hypothesis. These problems are usually conjectures (sort of like a hypothesis in science) meaning that they appear to be true for every number, but we have not been able to prove it. Testing every number is impossible since there are infinitely many numbers so mathematicians use mathematical rules to try to come up with a proof and show that the conjecture is true for for all numbers. An example of this is the famous Fermant's Last Theorem that no three positive integers a, b, and c satisfy the equation a^n + b^n = c^n for any integer value of n greater than two. While it appeared to be true for all numbers it took mathematicians 400 years to prove that it is using some very complex mathematics.. These problems often do not have obvious real world applications, however the our understanding of the physical world tends to play catch up with mathematics meaning that they may very well be discovered to have physical meanings in the future. That being said there are math problems like the Continuum Hypothesis which appear to be truly unsolvable and they arise from the axioms of mathematics.

It's pretty simple. Someone wants to know something. Let's take multiplying an even number by an odd number gives you an even number. (obviously this isn't an unsolved math problem, but it is easy to deal with and understand)

They realize that for every number they can think of this is true, yet they want to know if it is ALWAYS true that way it can be used in all math problems without checking. Until someone can figure out a way to mathematically prove it, it is an unsolved problem.

The current great unsolved problems are just scenarios like this that for one reason or another no one has come up with a proof for yet.

Most of you are referring to unsolved problems. The answers are great for that question. However, there's an interesting, alternative interpretation of "unsolvable".

"Unsolvable" problems is, interpreted very literally, a different class of problems altogether. They are problems that we've determined (conclusively) to have no consistent solution beyond relying on an additional "primitive" assumption.

One well-known problem that was later proven to be unsolvable is "How do we axiomatically define sets in a way that allows for their usage in modern mathematics?" Traditionally, the ZFC axioms were considered "necessary" for mathematics to be easy to use for applications, but some of them are unintuitive, so perhaps there is an alternative set of more basic axioms that allow us to define the whole system. As it turns out, this is unsolvable:

if ZFC is consistent, it is impossible to axiomatize ZFC using only finitely many axioms

-

Another closely related problem is how to define cardinality of infinite sets. As it turns out, the statement...

Is there a set whose cardinality is strictly between that of the integers and the real numbers? (CH)

...cannot be proved or disproved in a world with ZFC axioms. It is possible to to think of a ZFC-consistent world where such a set can exist, and it is also possible to think of a ZFC-consistent world where such a set can never exist.

The more general mathematical term is to say "ZFC and CH are independent". This term is preferred because it indicates the fundamental symmetry of unprovable propositions: in a world where you assume CH as primitive, but not all of the ZFC axioms are assumed, you wouldn't be able to determine if all the ZFC axioms are valid.

These results aren't as deep as they sound; rather, they simply indicate that a prior object/axiom was not rigorously defined. In a sense, each "fundamental" problem that is unsolvable is associated with a "dual" problem that becomes solved as a result: whether the definitions used in the statement of the fundamental problem are exhaustive ways to define a system.

  • If a "fundamental" problem is proven to be unsolvable, we have solved the "dual" problem by showing that the definitions do not cover all aspects of a system.

  • If a "fundamental" problem is solved, we haven't yet solved the "dual" problem of establishing whether the definitions are exhaustive.
